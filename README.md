## 기술 스택
- React
- MSW
- styled-components
- Sass
- react-router
- Axios

## 구동방법
현재 백엔드는 구현되지 않았습니다. 프론트 프로젝트에서 MSW를 사용해 모의 데이터를 받을 수 있습니다.

먼저, node_modules를 설치해야합니다. 이 프로젝트는 yarn 으로 동작합니다.
```
yarn install
```
이 후, 로컬에서 실행하기 위해 `yarn start`를 입력합니다.
```
yarn start
```
